How to define a batch file (windows)

If you sim with the same set of parameters multiple times you should consider creating a batchfile. Windows uses .bat-files. Once defined they can be used like a normal command. You can use a normal texteditor to edit a batchfile. In this example a batch file simbatch1.bat is created with the echo command. Next line shos how to use simbatch1.bat

>echo tu_optimize %1 "eerie-spam;nbd-spam;mydeck" -r sim 1000 > simbatch1.bat
>simbatch1 mydeck

History of tu_optimize

I play and played flash tyrant for several years and the original tyrant_optimze was the fastest optimizer and highly used by all players. When tyrant unleashed started I had a look at the source code of tyrant_optimize and figured it would take something in the ballpark of 20 hours to adjust the sources to work for tyrant unleashed. Unfortunatly I was very busy and so the project was put on hold.

Some weeks later I had to learn git for another project so I was looking for a none trivial example to work with git and github. tyrant_optimize came to my mind again and so I started with this project. Effort till first version 0.8 (first alpha) as arounf 24 hours so my original guess was pretty accurate.
