How to define a batch file (windows)

If you sim with the same set of parameters multiple times you should consider creating a batchfile. Windows uses .bat-files. Once defined they can be used like a normal command. You can use a normal texteditor to edit a batchfile. In this example a batch file simbatch1.bat is created with the echo command. Next line shos how to use simbatch1.bat

>echo tu_optimize %1 "eerie-spam;nbd-spam;mydeck" -r sim 1000 > simbatch1.bat
>tu_optimize mydeck "eerie-spam;nbd-spam;mydeck" -r sim 1000
